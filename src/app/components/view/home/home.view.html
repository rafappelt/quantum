O Objetivo dessa aplicação é demonstrar a implementação de uma arquitetura básica de um projeto de front-end.

As principais tecnologias utilizadas foram: Angular.js, HTML5 e SCSS.

Ao clicar no menu "IBOVESPA", uma interface REST é acessada e o histórico de fechamento da Ibovespa é exibido de forma gráfica em um SVG.

Passando o mouse sobre os picos do gráfico é possível ver os valores de abertura, fechamento, máximo e mínimo.

O Código-fonte pode ser avaliado no repositório do github
